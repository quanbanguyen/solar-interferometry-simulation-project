Hướng dẫn sử dụng

  • Load file SWS: File .txt gốc từ Learmonth (định dạng RSTN HHMMSS, 9 cột). Phần ban đêm (//////) tự động bị loại.
  • Chọn tần số: Nhấn vào các ô MHz để chọn cột dữ liệu từ file SWS (245–15400 MHz).
  • Anten: Nhập vị trí East/North từng anten (mét). Preset "compact EW" = hệ 4 anten dự án [0,8,16,24] m.
  • Tần số sóng mang: Kéo slider để thấy ảnh hưởng đến λ, chu kỳ vân, và độ phân giải.
  • Declination δ: Thay đổi xích vĩ nguồn ảnh hưởng đến dạng quỹ đạo UV và thời gian transit.
  • Tab Hình ảnh: Nhấn "Render" để tính PSF và dirty image (chậm hơn). Dirty image dùng biên độ flux làm trọng số.

Chuyển đổi Primary Beam .npz → JSON

Chạy đoạn Python sau để xuất file beam.json dùng được trong tool này:

import numpy as np, json

d = np.load("UFRAME_primary_beam_610MHz.npz")
# Keys: theta_deg (181,), phi_deg (361,), gain_dBi (181,361)
# Convention: elevation = -theta_deg  (theta=-90 → el=90°, theta=0 → el=0°)
theta   = d["theta_deg"]
gain_2d = d["gain_dBi"]    # shape (181, 361)

# Bán cầu trên: el 0..90°
mask        = (theta >= -90) & (theta <= 0)
gain_upper  = gain_2d[mask, :]
gain_dBi_1d = gain_upper.max(axis=1)   # envelope: max theo phi
el_deg      = (-theta[mask]).tolist()  # [90, 89, ..., 0]

# Chuẩn hóa tuyến tính 0..1
gain_max = gain_dBi_1d.max()
gain_lin = (10 ** ((gain_dBi_1d - gain_max) / 10)).tolist()

out = {
    "theta_deg": el_deg,
    "gain":      gain_lin,
    "gain_dBi":  gain_dBi_1d.tolist(),
    "freq_MHz":  float(d["freq_MHz"]),
    "note":      "1D elevation envelope (max over phi)"
}
with open("beam.json", "w") as f:
    json.dump(out, f, indent=2)
print(f"Saved beam.json  ({len(el_deg)} pts, peak={gain_max:.2f} dBi)")

Công thức tham khảo

  • Bước sóng: λ = c / f
  • Toạ độ UV (EW baseline): u = B sin H / λ, v = −B sin δ cos H / λ
  • Chu kỳ vân tại transit: T = λ / (B cos δ ω_E) giây
  • Độ phân giải: θ ≈ λ / B_max radian
  • Jinc visibility (đĩa đồng nhất): |V| = |2J₁(π θ☉ b/λ) / (π θ☉ b/λ)|
  • Null đầu tiên (Mặt Trời 0.53°): B_null ≈ 1.22λ / θ☉ ≈ 130 m tại 610 MHz
